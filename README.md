# 💼 Proyecto: Testing Unitario de SmartWallet

**Desarrollador:** Sergio Rodriguez
**Tecnologías:** Java 17 | Maven | JUnit 5 Jupiter

---

### 📋 Resumen
Este repositorio contiene la implementación de una billetera virtual (`SmartWallet`) y su respectiva suite de pruebas unitarias. Se aplican reglas estrictas de validación para transacciones, incluyendo un sistema de recompensas (cashback) y límites de saldo basados en el nivel de membresía del usuario.

### ⚙️ Instrucciones de Ejecución
El proyecto está gestionado con Maven. Para validar la cobertura y el estado de las pruebas, ejecute el siguiente comando en su terminal:

```bash
mvn test
